# Enlarge-Web
Enlarge-Web是Enlarge系统PC浏览器操作手机的前端界面项目,此项目

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
